Log Diff

I interned on the OMS Service Reliability Engineering team in a team of three interns. Our project was a Java application that visualized and compared log output between WATS runs using Grafana, InfluxDB, Logstash and Graphite. My work was primarily on Logstash deployment, using the Logstash API to read logs from an OMS.


I supported both WebLogD (which is deployed to all Workday SUVs and is still in use today) and the Log Diff application. Work was conducted in two week sprints, using JIRA to track bugfixes and development work.
